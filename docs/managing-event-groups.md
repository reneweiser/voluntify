# Organizing Event Groups

Event groups let you organize related events under a shared public page. Use them for multi-day festivals, recurring series, or any set of events you want volunteers to discover together.

## Create an Event Group

1. Click **Events** in the sidebar.
2. Click the **Event Groups** tab.
3. Click **Create Group**.
4. Fill in:
   - **Group Name** -- A name for the group (e.g., "Summer Festival 2026").
   - **Description** -- Optional details about the group, shown on the public page.
   - **Title Image** -- Optional hero image (JPG, PNG, or WebP, max 2 MB).
5. Click **Create**.

**Who can do this**: Organizer only.

## Edit Group Details

1. Click **Events** in the sidebar, then the **Event Groups** tab.
2. Click the group you want to edit.
3. Click **Edit**.
4. Update the name, description, or title image.
5. Click **Save**.

To remove a title image, click **Delete Image** while editing.

**Who can do this**: Organizer only.

## Assign Events to a Group

There are two ways to assign an event to a group:

**From the group detail page:**

1. Open the event group.
2. Use the **Add Event** dropdown to select an ungrouped event.
3. The event is added immediately.

**From the event edit form:**

1. Go to an event's **Overview** page and click **Edit**.
2. Select a group from the **Event Group** dropdown.
3. The assignment takes effect immediately.

Only ungrouped events appear in the dropdown. To move an event to a different group, remove it from its current group first.

**Who can do this**: Organizer only.

## Remove Events from a Group

**From the group detail page:**

1. Open the event group.
2. Click the remove icon next to the event you want to ungroup.

**From the event edit form:**

1. Go to the event's **Overview** page and click **Edit**.
2. Set the **Event Group** dropdown to the empty option.

Removing an event from a group does not delete the event -- it simply ungroups it.

**Who can do this**: Organizer only.

## Share the Group Page

Each event group has a public URL at `/groups/{publicToken}`. To share it:

1. Open the event group.
2. Copy the public URL shown on the group detail page.

The public page shows:
- The group name, description, and title image.
- Cards for each **published** event in the group, ordered by start date.
- Each card shows the event name, dates, location, volunteer count, and a link to the event's signup page.

Draft and archived events are not shown on the public page. Publish your events before sharing the group link.

## Delete an Event Group

1. Open the event group.
2. Click **Delete**.
3. Confirm the deletion.

Deleting a group ungroups all its events -- the events themselves are not deleted.

**Who can do this**: Organizer only.

## Tips

- **Publish events before sharing the group link** -- Only published events appear on the public page.
- **Add a title image** -- A hero image makes the public page more inviting and is used for social media previews.
- **Use descriptive group names** -- Volunteers see the group name on the public page, so make it clear what the group is about.
- **Cloned events start ungrouped** -- If you clone an event that belongs to a group, the clone is not automatically added to the same group.
